# Vayal Agro - Autonomous Agriculture & Warehouse Robot System

A full-stack web application showcasing the Vayal Agro autonomous robot system for vegetable cultivation, smart transportation, and intelligent warehouse placement.

## Project Overview

This system demonstrates an advanced robotics project featuring:
- Autonomous vegetable cultivation
- Smart crop transportation using specialized containers
- Line sensor-based warehouse navigation
- Intelligent placement and warehouse management

## Architecture

The project consists of two independent applications:
- **Frontend**: React application with professional UI
- **Backend**: Node.js API with MongoDB integration

## Quick Start

### Prerequisites
- Node.js (v16 or higher)
- MongoDB Atlas account
- Git

### Installation

1. **Clone and navigate to project:**
```bash
cd "C:\Users\LENOVO\OneDrive\Documents\Custom Office Templates\en_thara_web"
```

2. **Setup Backend:**
```bash
cd backend
npm install
npm run dev
```

3. **Setup Frontend (in new terminal):**
```bash
cd frontend
npm install
npm run dev
```

4. **Access Application:**
- Frontend: http://localhost:3000
- Backend API: http://localhost:5000

### Frontend
- React 18 with Vite
- Tailwind CSS
- React Router DOM
- Axios

### Backend
- Node.js with Express
- MongoDB with Mongoose
- CORS enabled
- Environment configuration

## Database Configuration

The backend is configured to use MongoDB Atlas with the provided connection string. The database and collections will be automatically created when data is first inserted.

## Deployment

Both frontend and backend are designed for independent deployment:
- **Frontend**: Can be deployed to Vercel, Netlify, or any static hosting
- **Backend**: Can be deployed to Heroku, Railway, or any Node.js hosting
